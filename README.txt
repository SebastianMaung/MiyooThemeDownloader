# MiyooThemeDownloader
Theme browser and downloader for the Miyoo Mini Plus
Add these to your App folder of your Miyoo, and run the "Online Theme Shop".
Let it download all the previews of the themes, then it will have you in the file explorer. 
Select a theme preview, then press left or right to naviagate through.
If you find one you like, then press b and take note of the number of the preview the cursor is on. (preview.**19**.png)
Then press menu, then exit. It will ask your for the ID, and enter the number (Press x if there is no keyboard) and press start.
It should start downloading, and then you will be sent to the theme selector.
